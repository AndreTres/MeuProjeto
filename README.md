# 🎰 Betano - Casa de Apostas

Um projeto inspirado no site da casa de apostas Betano, desenvolvido com HTML, CSS e JavaScript puro.

## 🚀 Características

- **Design Moderno**: Interface inspirada na Betano com cores laranja (#FF4B00) e cinza escuro (#1C1C1C)
- **Totalmente Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Interativo**: Slider automático, menus dropdown, animações e efeitos hover
- **Performance Otimizada**: Código limpo e otimizado para carregamento rápido

## 📁 Estrutura do Projeto

```
MeuProjeto/
├── index.html          # Página principal
├── styles/
│   └── style.css       # Estilos CSS
├── scripts/
│   └── main.js         # JavaScript interativo
├── assets/             # Pasta para imagens e ícones
└── README.md          # Documentação
```

## 🎨 Funcionalidades

### Interface
- ✅ Cabeçalho fixo com logo e navegação
- ✅ Banner principal com slider automático
- ✅ Cards de jogos organizados em grid responsivo
- ✅ Seção de promoções especiais
- ✅ Rodapé com links e informações legais

### Interatividade
- ✅ Slider automático com controles manuais
- ✅ Menu mobile responsivo
- ✅ Animações de scroll e hover
- ✅ Modais de login e cadastro
- ✅ Efeitos parallax
- ✅ Smooth scroll entre seções

### Responsividade
- ✅ Desktop (1200px+)
- ✅ Tablet (768px - 1199px)
- ✅ Mobile (até 767px)

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Flexbox, Grid, Animações, Media Queries
- **JavaScript ES6+**: Classes, Arrow Functions, Async/Await
- **Font Awesome**: Ícones
- **Google Fonts**: Tipografia (Roboto)

## 🚀 Como Usar

1. Clone ou baixe o projeto
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Navegue pelas seções e teste as funcionalidades

## 📱 Recursos Mobile

- Menu hambúrguer responsivo
- Touch gestures no slider
- Layout otimizado para telas pequenas
- Botões e textos redimensionados

## 🎯 Funcionalidades JavaScript

### Slider Automático
- Troca de slides a cada 5 segundos
- Controles manuais (anterior/próximo)
- Indicadores de posição (dots)
- Pausa automática no hover

### Animações
- Fade in/out nos elementos
- Hover effects nos cards
- Parallax no banner
- Smooth scroll entre seções

### Modais
- Sistema de modais para login/cadastro
- Animações de entrada/saída
- Fechamento por clique fora ou botão X

- CSS e JS organizados em módulos

## 🔧 Personalização

Para personalizar o projeto:

1. **Cores**: Edite as variáveis CSS no arquivo `style.css`
2. **Conteúdo**: Modifique o HTML em `index.html`
3. **Funcionalidades**: Adicione novas classes JavaScript em `main.js`
